# Z3 Glow Skincare Set

**Kategori**: Kecantikan / Skincare
**Harga**: Rp 299.000 (set lengkap)
**SKU**: TL-SK-007

## Isi Set (4 Produk)
1. **Gentle Cleanser** (100ml) - Sabun muka pH balanced, cocok semua jenis kulit
2. **Hydrating Toner** (120ml) - Toner pelembap dengan Hyaluronic Acid dan Centella
3. **Brightening Serum** (30ml) - Serum pencerah dengan Niacinamide 10% + Vitamin C
4. **Moisture Barrier Cream** (50ml) - Pelembap dengan Ceramide untuk menjaga skin barrier

## Detail Bahan Utama
- Hyaluronic Acid: melembapkan dan menahan air di kulit
- Niacinamide 10%: mencerahkan, mengecilkan pori, kontrol minyak
- Vitamin C (Ascorbyl Glucoside): antioksidan, mencerahkan
- Centella Asiatica: menenangkan kulit sensitif dan iritasi
- Ceramide: memperkuat skin barrier

## Cocok Untuk
- Semua jenis kulit (normal, kering, berminyak, kombinasi)
- Kulit kusam yang butuh pencerah
- Usia 18+
- Aman untuk ibu hamil (tidak mengandung retinol)

## Cara Pakai
1. Cleanser: Gunakan pagi dan malam, pijat lembut di wajah basah, bilas
2. Toner: Tuang ke kapas atau telapak tangan, tepuk lembut ke wajah
3. Serum: 3-4 tetes, ratakan di wajah setelah toner, tunggu menyerap
4. Cream: Ambil secukupnya, ratakan sebagai langkah terakhir
